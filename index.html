Samurai Pro (Automação RBI)

Samurai Pro

Reduzi o consumo de RAM de 900MB para menos de 60MB por sessão em automação web utilizando Kernel Samepage Merging (KSM) no Linux e orquestrando fluxos em Python e n8n.

O Problema: Bloqueios constantes de WAFs (Cloudflare/Akamai) e alto custo de RAM inviabilizando Remote Browser Isolation em escala.

Solução: Arquitetura de infraestrutura Python/Docker focada em KSM e substituição de seletores DOM frágeis por inferência visual auto-curável (Auto-healing) via GPT-4o Vision.

Python / Linux KSM GPT-4o Vision

WSP Finance (Cloud FinOps)

WSP Finance SaaS

Reduzi o OPEX de nuvem em 28% construindo um Cofre Digital de notas fiscais via Cloudflare R2 com arquitetura de taxa zero de egresso de dados (Zero Egress Fee).

VIVA Telemetria

Blindei o Event Loop da API contra falhas assíncronas em telemetria clínica, aplicando o padrão Transactional Outbox com filas BullMQ e Redis (NestJS).
